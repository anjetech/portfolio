Project Preview

Digital Insurance Management System

JavaFX Desktop App + MySQL (JDBC)

A JavaFX desktop application that captures customer and policy details, calculates insurance premiums based on policy type, stores records in a database, and displays saved policies in a TableView. Built with clean UI structure, event-driven logic, and database integration.

JavaJavaFXJDBCSQL / MySQLTableView
View Screenshots GitHub

What it does

  • Captures customer and policy info in a JavaFX form
  • Calculates premium automatically from coverage + policy type
  • Saves customers and policies into a SQL database

Key features

  • Policy type selection (Life / Health / Auto)
  • Premium % logic (5% / 8% / 10%)
  • View Policies button loads DB data into TableView

What I learned

  • Building JavaFX forms and wiring events cleanly
  • Connecting Java to SQL using JDBC (PreparedStatements)
  • Displaying data with TableView + ObservableList

Build Notes

Architecture

JavaFX UI built with GridPane + form inputs, wired with button handlers. Uses a Policy model (SimpleStringProperty) to bind data into a TableView.

Data

Uses JDBC with PreparedStatements to insert customers and policies into separate tables. The View Policies action joins the tables and loads results into an ObservableList for display.
