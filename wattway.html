Project Preview

WattWay

EV Charging & Load-shedding Mobile App

A mobile app that helps EV users find charging stations on a map and view load-shedding schedules. It includes secure login/registration, role-based access, profile management, and admin controls, with strong error handling and smooth screen navigation.

AndroidFirebase AuthFirestoreGoogle Maps EskomSePush APIMVVM
View Screenshots GitHub

What it does

  • Shows EV charging stations on a map
  • Displays load-shedding schedules (API-driven)
  • Profile management + admin controls

Key features

  • Secure login & registration (Firebase Authentication)
  • Role-based access using Firestore rules
  • Error handling + loading states for API responses

What I learned

  • MVVM structure for cleaner, maintainable code
  • NoSQL design + real-time syncing with Firebase
  • UI consistency and testing to reduce bugs

Build Notes

Architecture

Designed to integrate multiple services smoothly (authentication, database, maps, and external APIs), while handling asynchronous data flow, navigation stability, and clean maintainable structure using MVVM.

Data

Uses Firestore (NoSQL) with role-based access control and real-time syncing. Improved reliability by adding error handling/loading states and tightening access rules.
