F1 Reaction Time App

Interactive Web App

A browser-based reaction time game inspired by F1 lights-out starts. Users click as soon as the screen changes, and the app measures reaction speed in milliseconds. Built with clean UI, DOM updates, and precise timing logic.

HTMLCSSJavaScriptDOM
View Screenshots GitHub

What it does

  • Starts a reaction test with a random delay
  • Measures click speed in milliseconds
  • Shows results instantly (and resets to play again)

Key features

  • Randomised start timing to prevent guessing
  • Clear UI states (waiting, ready, clicked)
  • Basic error handling (too early clicks)

What I learned

  • Working with timers and accurate time measurement
  • DOM manipulation for live UI updates
  • Keeping logic clean with simple state handling

Build Notes

Logic

Uses a random delay before switching to the “GO” state, then records the time difference between state change and user click to calculate reaction time.

UI

UI updates are handled through DOM manipulation to reflect each state clearly (ready → waiting → go → result) for a smooth user experience.
