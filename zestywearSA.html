Project Preview

Zesty Wear SA

Consumer-to-Consumer (C-2-C) E-Commerce Website

A consumer-to-consumer (C-2-C) fashion marketplace that allows users to browse products by category, register and log in, and list items for sale. Built with a database-backed structure for storing products/users and managing user actions across the site.

HTMLCSSJavaScriptPHPMySQL
View Screenshots GitHub

What it does

  • Browse fashion items and view product details
  • Explore products by category for easier discovery
  • Create an account, log in, and list items for sale

Key features

  • User registration + login (user identification)
  • Product listings with category browsing (and search if included)
  • Database-driven structure for storing products and users

What I learned

  • How front-end and back-end work together in a real web app
  • Building with PHP + MySQL to create dynamic pages from stored data
  • Problem-solving through real development challenges and debugging

Build Notes

Architecture

Designed as a C-2-C marketplace with structured pages and PHP-driven logic to support browsing, authentication (register/login), and selling workflows.

Data

MySQL database used to store and retrieve core site data (such as users and products), supporting dynamic product listings and user-specific actions.
